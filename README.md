## Getting Started

1) Run the development server: 

```
i) cd server
ii) npm start.
'''

2) Run client on a different terminal:.

'''
i) cd client
ii) nmp start

'''

Open [http://localhost:3000](http://localhost:3000) with your browser on two separate tabs/windows and chat.
